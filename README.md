# traffic-light

Traffic Light Simulator

A simple interactive traffic light simulator built with HTML, CSS, and JavaScript.
Click the buttons to control the red, yellow, and green lights and learn how traffic lights work.

Features

- Interactive traffic light – STOP, READY…, GO! buttons control the lights.

- Responsive layout – works on desktop and mobile.

- Instructions panel – shows how to use each button.

- Clean, modern design – subtle shadows, rounded panels, readable text.
